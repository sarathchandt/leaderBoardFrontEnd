# Leaderboard System

A leaderboard system that allows users to enter their scores and analyze their position on the leaderboard. This project is built with React and Vite for a fast and efficient development experience.



## Technologies Used
- **Frontend**: React, Vite
- **Backend**: Express js
- **Database**: mongoDB
